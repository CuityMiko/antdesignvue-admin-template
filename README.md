
# G端主线 Ant Design Vue Admin Template For Vue2.x

### 使用说明

#### 1. 拉取项目代码

```
git clone 
```

#### 2. 安装依赖

* 推荐方式

```
yarn 
```
* 备选方式

```
npm i
```

```
cnpm i
```

#### 3. 运行项目
```
npm run dev
```

#### 4. 打包项目

正式环境：
```
npm run build-prod
```

测试环境：
```
npm run build-test
```

#### 5. 打包分析

```
npm run report
```

#### 6. 代码格式修复

```
npm run lint-fix
```
